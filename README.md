# myrepo_test1

這是一個示範專案，用於展示如何將本地專案上傳至 GitHub。

## 檔案說明

${PROJECT_NAME}_demo.sh\: 系統資訊顯示腳本
\.gitignore\: Git 忽略檔案配置
## 使用方式

克隆此倉庫：
\\bash
git clone https://github.com/zero2005s/myrepo_test1
\
進入專案目錄：
\\bash
cd myrepo_test1
\
執行示範腳本：
\\bash
./myrepo_test1_demo.sh
\
## 注意事項

確保您有適當的執行權限
腳本需要在 Linux/Unix 環境下執行
